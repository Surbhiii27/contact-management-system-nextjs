# Contact-management-system-nextjs
Frontend Contact Management Application with Create, Read, Update, and Delete (CRUD) functionality using Next.js.

## Features

- Add new contacts
- View contact details
- Update existing contacts
- Delete contacts
- Responsive user interface
- Fast performance with Next.js
- Clean and intuitive design

## Technologies Used

- Next.js
- HTML5
- CSS3
- JavaScript / TypeScript
```

## CRUD Operations

### Create
Add new contacts with required details.

### Read
View all saved contacts in a structured format.

### Update
Edit contact information whenever needed.

### Delete
Remove contacts from the contact list.

## Installation

1. Clone the repository

2. Navigate to the project folder

3. Install dependencies

```bash
npm install
```

4. Run the development server
    T1 -Run Json server(database) -npm run server
    T2- Run the project - npm run dev

5. Open your browser and visit

```text
http://localhost:3000
```
